## Objetivos

Al finalizar esta actividad, el estudiante:

* Establece el sistema de proyección de coordenadas en un mapa geográfico.
* Proyecta al vuelo capas geográficas desde diferentes sistemas de proyección.
* Crea rótulos dinámicos básicos y avanzados.
* Representa capas vectoriales por múltiples métodos de visualización.
* Crea filtros básicos y avanzados a partir de los atributos contenidos en una capa.
* Crea gráficos de una o múltiples variables y matrices de dispersión dinámicas para evaluar datos atípicos.
